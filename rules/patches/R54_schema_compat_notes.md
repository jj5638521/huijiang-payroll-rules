# R54.1 Schema Compat Notes（Attendance 施工说明字段）

## 背景
在不同施工表模板中，同一业务含义可能以不同表头导出：`施工说明`、`问题描述`、`其他说明`，以及部分系统中的备注型列（`报销说明`/`摘要`/`备注`）。若仅按单一列名识别，容易出现 MealForm 单选值漏命中。

## 本次兼容内容
- 扩展 AttendancePipe 的施工说明字段同义映射，覆盖：
  - `施工说明`
  - `问题描述`
  - `其他说明`
  - `报销说明` / `摘要` / `备注`（仅 AttendancePipe 范围内的施工表备注列）
- 增加表头清洗说明：
  - 去空白/制表符/全角半角/隐藏换行后匹配。
  - 对带括号扩展名的表头（如“问题描述（如设备损坏、特殊事项等）”）允许按关键词“问题描述”命中。

## 影响边界
- 这是字段识别补强，不改变 R54 业务逻辑、优先级和计算口径。
- 不影响 PaymentPipe：PaymentPipe 仍只使用支付记录表及其文本列规则；“已结清”等关键词判定范围不变。
